# IISのカレントフォルダ

IISの実行時のカレントフォルダは以下の用になる

デバッグ時
```
C:\Program Files (x86)\IIS Express
```

IISから起動時
```
c:\windows\system32\inetsrv
```

ファイルを取る時は注意されたし
