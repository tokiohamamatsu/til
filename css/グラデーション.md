# グラデーション

```css
/*上から下のグラデーション*/
#sample {
  background: -moz-linear-gradient(top, #FFC778, #FFF);
  background: -webkit-linear-gradient(top, #FFC778, #FFF);
  background: linear-gradient(to bottom, #FFC778, #FFF);
  /*グラデーション以外は省略*/
}
```
