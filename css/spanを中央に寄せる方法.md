# spanを中央に寄せる方法

span要素はtext-align: centerではセンタリングが出来ない。

span要素をセンタリングする方法はいくつかあるが、spanにクラスを設定して下記のようにするのが一番簡単だと思う。

```css
.center{
  display: block;
  text-align: center;
}
```
