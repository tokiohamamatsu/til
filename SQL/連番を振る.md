# 連番を振る

SQL Serverで連番を振るにはROW_NUMBER関数を使う

```sql
ROW_NUMBER() OVER ( [ PARTITION BY [パーティションカラム], [パーティションカラム], ...] ORDER BY [ソートカラム], [ソートカラム], ... )
```

例

```sql
SELECT 
  ROW_NUMBER() OVER(ORDER BY Age ASC) Num
, Name
, Age
FROM users
```

Ageが小さい順に連番を振っている。

|NUM|NAME|Age|
|---|---|---|
|1|Yamada Taro|10|
|2|Suzuki Ichiro|20|
|3|Sato Taro|30|
|4|Tanaka Jiro|40|
