# jquery.validationEngine 

## 表示するエラーメッセージを一つにする

```js
$("#form_id").validationEngine({
    maxErrorsPerField: 1
});
```

フォームにバリデーションをバインドする時、`maxErrorsPerField`に表示するエラーメッセージの最大数を指定する
