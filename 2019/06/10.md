## Visal Studio Code insidersのインストール方法

１．インストローラのダウンロード
　　・ダウンロードサイトにアクセス
２．セットアップウィザードを起動
　　・同意して設定は個人の自由
３．Visual Studio Code - Insidersを起動


## GitHubの登録

１．アカウント作成
　　・メールアドレスは会社のを使用
２．リポジトリの作成
　　・privateを選択
３. Visal Studio Code にリポジトリを持ってくる
　　・usernameとpasswoldが求められるので忘れないように
４. Extensionsから便利なものをインストール
　　・Markdown Preview Enhancedなど
