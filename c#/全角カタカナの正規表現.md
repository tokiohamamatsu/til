# 全角カタカナの正規表現

c#での正規表現

```c#
using System.Text.RegularExpressions;

if (!Regex.IsMatch(Data.カナ, "^[ア-ー]+$")) {
    ErrorProvider.SetError(カナTextBox,"カタカナで入力してください");
    return;
}
```

上記でカタカナの正規表現に成功

c#の正規表現はRegexクラスのIsMatchを使い行なう。

Unicodeのコード表は意味わからん
