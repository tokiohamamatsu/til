# Entity Frameworkでテーブルから列を取得

Entity Frameworでテーブルからコンボボックスに値を入れる

やり方としてはLinqのSelectを使いデータを絞り込み、それをToArrayで配列にした後
コンボボックスのDatasourceに入れる。
