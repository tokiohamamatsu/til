# テキストボックスへのデータバインディング

事前にBindingSourceをフォームに設定し、プロパティの設定を行う

バインドの手順

テキストボックスのDataBindingsプロパティを開き内部のTextのコンボボックスを開き、bindingSourceのインスタンス名のノードを開き、フィールドを選択する。
