# IDisposableとは

IDisposableはアンマネージドリソースを開放する方法を明示的に開放する方法を定義したものらしい

アンマネージドリソースは外部リソースのことで、ファイルやハンドルを指す。

CRL(.NETの動作環境)ではマネージリソースはGC(ガベージコレクション)が回収してくれるが、アンマネージドリソースはユーザーが明示的に開放する必要がある。

そのためにIDisposableを使う必要がある。
