# 配列の重複削除

```php
$satff=[];
foreach ($detas as $deta) {
            if (!array_key_exists($deta->id, $staff)) {
                $staff[$deta->id]=$deta->staff;
            }
        }
```

array_key_exists は指定したキーが配列にあるか調べる。
